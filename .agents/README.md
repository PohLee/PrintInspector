# Agent Instructions for ESC/POS Virtual Printer Project

This directory contains specialized agent instructions and skill definitions for working with the ESC/POS Virtual Printer Emulator project.


## Available Skills

- **`flutter_ui_ux_material_design`**: Best practices for Flutter UI/UX and Material Design 3.

## Usage

When working on this project, you can load these skills using the `/skill` command or by specifying them in agent configurations.

Example:
```
/skill flutter_ui_ux_material_design
```

These skills provide context-aware guidance for implementing features that follow Flutter and Material Design best practices.
